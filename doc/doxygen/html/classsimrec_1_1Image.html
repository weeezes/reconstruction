Detailed Description

Image reads and contains 8-bit black and white image data.

The Image class is used for reading 8-bit image data from a file and to keep track of it's information, mainly the width and height of the image.

void Image::upscaleToClosestPowerOfTwo ( )

Upscales the image.

Upscales the image to a rectangle, which has a power-of-two valued side length. The side length is determined from the maximum of the width and height of the image, so that the resulting side length is the smallest power of two value that is larger than this maximum. The original image data is padded with zeros, so that no resizing of the original data happens.
